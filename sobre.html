Sobre o Projeto

Este site foi desenvolvido como um exercício prático para aprimorar e demonstrar minhas habilidades em desenvolvimento front-end. O objetivo foi criar uma experiência de usuário agradável e responsiva, utilizando tecnologias modernas como HTML5, CSS3 e o framework Bootstrap 5. A escolha de um site de turismo para Campo Alegre, no Tocantins, foi inspirada pela beleza natural e potencial turístico da região.

Olá, eu sou Diego Aquino

Sou um entusiasta da tecnologia e estudante de desenvolvimento web, sempre em busca de novos desafios para aprimorar minhas habilidades. Tenho paixão por criar interfaces limpas, intuitivas e que proporcionem uma boa experiência para o usuário final.

Meu objetivo é me tornar um desenvolvedor front-end completo, dominando as principais ferramentas do mercado para construir soluções web inovadoras e eficientes. Este projeto é mais um passo na minha jornada de aprendizado contínuo.
